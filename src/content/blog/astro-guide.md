## 为什么选择 Astro

Astro 是一个现代化的静态站点生成器，默认输出零 JavaScript，性能极佳。

## 核心优势

- **零 JS 输出**：默认不包含任何客户端 JavaScript
- **多框架支持**：可以在同一项目中使用 React、Vue、Svelte 组件
- **内容优先**：以 Markdown 为中心的内容管理
- **岛屿架构**：只在水合需要的部分加载 JS

## 快速开始

```bash
npm create astro@latest
```

选择博客模板即可快速上手。
